# OPCUA_prac
Creating simple OPC UA Server/Client with python
---

## Installtion
clone this project:
```
git clone https://github.com/HYORIMLEE/OPCUA_prac.git
```
Install packages:
```
$ pip install -r requirements.txt
```
## What works?(client)
* browsing, reading and editing values
* getting nodes by path and nodeids
